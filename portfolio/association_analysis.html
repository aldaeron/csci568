Association Analysis

Introduction

Association analysis is the discovery of interesting relationships hidden in large data sets. Typically these data sets are market basket data. The result of the analysis is a set of rules in the form:

(Itemset1) => (Itemset2)

Rather than calculate every permutation of itemsets to find good rules, frequent itemsets are generated first. Non-frequent itemsets will have poor rule metrics. Good rules are determined using two rule metrics: support and confidence. Support is the number of times ItemSet1 AND ItemSet2 appear in the total number of records. Confidence is the number of times ItemSet1 AND ItemSet2 appear compared to the number of times ItemSet1 appears by itself . The heart of frequent itemset generation is the a priori principle which states that if an itemset is frequent then its subsets must also be frequent. This means that if a set is infrequent, its supersets must also be infrequent. Using this principle, infrequent small itemsets can be used to prune a lot of supersets in only a few steps. There are two efficient methods to build frequent itemsets Fk-1 X Fk-1 and Fk-1 X Fk1.
